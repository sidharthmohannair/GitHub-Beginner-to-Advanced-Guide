# Branching in Git

## Creating a Branch
Create a new branch.

```bash
git branch new-branch
```

## Switching Branches
Switch to a different branch.

```bash
git checkout new-branch
```

## Creating and Switching Branches
Create and switch to a new branch in one command.

```bash
git checkout -b new-branch
```

## Deleting a Branch
Delete a branch.

```bash
git branch -d new-branch
```
